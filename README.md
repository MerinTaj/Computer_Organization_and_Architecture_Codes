# Computer Organization and Architecture Codes

This repository contains Assembly Language programs developed for the Computer Organization and Architecture (COA) course.

## Tool Used

- EMU8086 Emulator

## How to Run

1. Install EMU8086.
2. Open EMU8086.
3. Select **File → Open** and choose the desired `.asm` file.
4. Click **Compile**.
5. Click **Emulate**.
6. Run the program to view the output.
